## Description

PANR - A Nestjs microservices project with features : Payments, Authorization, Notifications, Reservations

## Project architecture

![architecture drawio (1)](https://github.com/jha0402/panr/assets/67306416/a03558b5-28f6-4d44-8430-c65929957d7e)

## Installation

```bash
$ pnpm install
```

## Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Test

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
